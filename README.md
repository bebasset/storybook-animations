# 📚 Storybook Animations

Animated motion picture books for kids — designed for classroom use, YouTube read-alouds, and family storytime.

Each book is a **single self-contained HTML file** — no installs, no internet needed. Just open in any browser and go!

---

## 🎬 How to Use

### Watch / Present
1. Open the `animation.html` file inside any book folder
2. Use the **Next / Back buttons** or press the **→ / ← arrow keys** to turn pages
3. Read the matching script from `voiceover-script.md` as you advance through the scenes

### Record for YouTube
1. Open the animation in **Chrome** or **Firefox** (full screen with F11)
2. Record your screen using **OBS Studio** (free) — [obsproject.com](https://obsproject.com)
3. Read the voiceover script as you click through the pages
4. Export and upload to YouTube!

### Use in the Classroom
- Open the animation on a projector or smartboard
- Read aloud to students as you advance slides
- Works on **Chromebooks, Windows, Mac, iPad** — any device with a browser

## ➕ Adding a New Book

1. Create a new folder under `books/` (e.g. `books/my-new-book/`)
2. Add your `animation.html` file (see `books/_template/` for a starter)
3. Add a `voiceover-script.md` with your read-along text
4. Update the Books table in this README

---

## 💡 Tips for Great Read-Alouds

- **Pace yourself** — pause at dramatic moments and let kids look at the animation
- **Use different voices** for different characters
- **Don't rush the big moments** — the sneeze, the celebration, the ending lesson
- **Invite participation** — ask kids "What do you think happens next?"
- The full book takes about **6–8 minutes** at a relaxed read-aloud pace
